STANDORT-ID
So findest du deine STANDORT-ID:
1. Gehe zur Abfall-Seite der Stadt Dresden
2. Suche deine Adresse und klicke „Jetzt finden"
3. Rechtsklick (oder langer Tipp) auf den iCal-Link → „Link kopieren"
4. Füge den kopierten Link unten ein – die ID wird automatisch erkannt.
Oder: iCal-Link einfügen
Die STANDORT-ID wird automatisch aus dem Link extrahiert.
